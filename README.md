# Line Graph Neural Networks for Link Prediction



## Introduction

Pytorch implementation of ''**LMGATCDA: Graph neural network with labeling trick for predicting circRNA-disease associations**''.



Dependencies
--------------------------------

- python 3.6+
- Pytorch 1.5.0+
- dgl 0.6.0 +
- scipy
- numpy
- pandas
- sklearn
